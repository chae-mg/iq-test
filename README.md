# Simple IQ Test

Korean self-check IQ-style quiz built as a single static HTML page.

## Use

Open `index.html` in a browser, or deploy the repository with GitHub Pages.

The test asks for a name and birth date, runs 30 multiple-choice questions, and reports:

- total score
- approximate level
- category scores
- weakest area
- incorrect or unanswered questions

This is not an official IQ test.
