# Simple IQ Test

Korean self-check IQ-style quiz built as a single static HTML page.

## Use

Open `index.html` in a browser, or deploy the repository with GitHub Pages.

The test asks for a name and birth date, randomly selects 30 questions from an 80-question bank, and reports:

- total score
- approximate level
- rough school-age or age-level interpretation
- category scores
- weakest area

This is not an official IQ test.
